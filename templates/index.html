The Day Trader's Cheat Sheet

What is it?

This app is an experiment to see if machine learning could be used to help day traders beat the market. The app is built around a neural network designed to predict the closing price of a stock on any given day. The neural network learns from historical stock S&P 500 data to predict the closing price on the current day or the last day that the market was open. Day traders can use this information to determine whether or not they should buy a given stock based on how its price will change by the end of the day to maximize daily profits.

How do you use it?

If you are running the server through the command line, leave the command line window open. Enter the stock symbol of the company you would like to analyze below.
